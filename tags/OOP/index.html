    When I first learned F#, one of the designs I didn’t understand is that in F# you have to use a lot of pipeline operators ( |> ), while in other languages you just use dots ( . ). For example, for a C# expression like x.Select(n => ...) , you have to write x |> List.map (fun n -> ...) in F#, which looks verbose at the first sight.
